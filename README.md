## ✨ Features | 特性

- 🌐 **Bilingual Support | 中英文双版本** - Seamless language switching between Chinese and English
- 🎨 **Modern Tech Aesthetic | 科技感设计** - Dark theme with gradient accents and particle animations
- 📱 **Fully Responsive | 完全响应式** - Optimized for desktop, tablet, and mobile devices
- ⚡ **High Performance | 高性能** - Smooth animations and optimized loading
- 🎯 **AI-Focused Content | AI专业内容** - Showcases AI/ML expertise, projects, and achievements

## 🛠️ Tech Stack | 技术栈

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Styling**: Custom CSS with CSS Variables, Flexbox, Grid
- **Animations**: Canvas API (Particle System), CSS Animations, Intersection Observer
- **Icons**: Font Awesome 6
- **Fonts**: Inter, JetBrains Mono

## 🚀 Getting Started | 开始使用

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize the content to match your profile
4. Deploy to GitHub Pages or your preferred hosting

## 🔧 Customization | 自定义

### Personal Information | 个人信息
Edit the following in `index.html`:
- Name and title in the hero section
- About section details
- Project descriptions and links
- Contact information

### Content Translation | 内容翻译
All translatable content is in `script.js` under the `translations` object:
```javascript
const translations = {
    zh: { ... },  // Chinese
    en: { ... }   // English
};
```

### Styling | 样式
Modify CSS variables in `style.css`:
```css
:root {
    --primary: #6366f1;
    --secondary: #22d3ee;
    --accent: #f472b6;
    /* ... */
}
```
